# HydroChemix Water Treatment Chemicals Knowledge Base

This repository is a lightweight public mirror of practical water treatment chemical notes for buyers, engineers and plant operators.

Topics include PAC selection, PAM selection, activated carbon selection, jar testing, COA/SDS document checks, textile wastewater treatment, sludge dewatering and drinking water polishing.
